# Car Rental Management with CRUD
A Python-based Car Rental Management System developed using CRUD (Create, Read, Update, Delete) operations and a booking management feature.

## Project Overview
This project is a console-based application that helps manage a car rental business. Users can view available cars, add new cars, update existing data, delete records, make bookings, and process payments.

## Features
### Read
- Display all rental cars
- Filter by fuel type
- Filter by transmission
- Filter by maximum rental price

### Create
- Add new car data
- Auto-generate car codes

### Update
- Edit specific car information
- Edit all car information

### Delete
- Remove car data from the system

### Booking System
- Rent one or multiple cars
- Automatic stock checking
- Shopping cart system

### Payment System
- Total cost calculation
- Payment validation
- Change calculation
- Automatic stock update after payment

## Technologies Used
- Python
- Dictionary
- List
- Functions
- Loops
- Conditional Statements
